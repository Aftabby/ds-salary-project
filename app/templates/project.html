Data Scientists Salary Prediction

Scraping data from Glassdoor and building a model to predict salaries for various roles of data scientists.

{% endblock %} {% block section %}

1. Introduction

Welcome! This project involves a comprehensive analysis of data scientist job listings scraped from Glassdoor. Initially, raw data was collected, capturing every possible detail about the job listings, followed by data cleaning and Exploratory Data Analysis (EDA), including feature encoding.

Several predictive models were applied, including

  • Linear Regression
  • Lasso Regression
  • Random Forest

The Random Forest model, which demonstrated the highest accuracy, was selected.

To showcase the results, a web application was developed using Flask and deployed within a Docker container. All project files and detailed documentation are available on my GitHub profile.

2. Problem Statement

Understanding job market trends for Data Scientist roles is crucial for both job seekers and employers.

This project aims to address the following key questions:

  • Identifying Key Influencers: What factors (such as, location, required skills, company size, and industry etc.) influence salary estimates for Data Scientist roles?
  • Analyzing Salary Variations: How do company size, location, and industry correlate with job salary variations?
  • Most Demanding Skill: Which skills and qualifications are most frequently listed in high-paying job postings?
  • Predictive Modeling: Can we build a predictive model to estimate salaries based on job features?
  • Interactive Visualization: Visualing the data in an interactive manner using Power BI Dashboard.
